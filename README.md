# Project CRUD ICC (Informatics Coding Camp)
## Tech Stack
#### Backend
- Go (Golang)
- Fiber
#### Frontend
- React
- Vite
- Tailwind

## How to clone this repository
**Clone With HTTPS**
> `git clone https://github.com/azrilpramudia/crud-react-go.git`
**Clone With SSH**
> `git clone`
## Install Dependencies
> `npm install`
## Run on Local (for development)
> `npm run dev`
